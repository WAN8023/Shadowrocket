## 二、填报Py

### #Example

```python
import requests
#导入模块
url="http://bbs.zcbxw.com/mag/xxx/xx/......."		#自行补全
headers={"Host":"bbs.zcbxw.com","xxx":"xxx","xx":"xx"}
data="phone=手机号码&country_code=86"
requests.post(url=url,headers=headers,data=data)
#将headers与url按抓包数据补全，其他代码不变，运行，就完成了一个请求
```

## 三、Bark通知py

```python
import requests
import json

url="https://api.day.app/push"
headers={
    "Content-Type": "application/json; charset=utf-8"}
data=json.dumps({               # 使用json{}不行，要使用json.dumps({})   #要把data进行json编码
    "body": "Test Bark Server",
    "device_key": "Xehm***********HvCL",
    "title": "bleem",
    "category": "category",
    "sound": "minuet.caf",
    "badge": 1,
    "icon": "https://day.app/assets/images/avatar.jpg",
    "group": "test",
    "url": "https://mritd.com"})
r=requests.post(url=url,headers=headers,data=data)
if r.status_code == 200:
    print("bark 推送成功！")
else:
    print("bark 推送失败！")
```

> 通过python模拟Post请求进行Bark推送。
